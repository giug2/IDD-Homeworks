Causal Feature Selection Method for Contextual Multi-Armed Bandits in Recommender System

Zhenyu Zhao zzhao@roblox.com RobloxSan MateoCaliforniaUSA Â andÂ  Yexi Jiang hjiang@roblox.com RobloxSan MateoCaliforniaUSA
(2024)
Abstract.

Features (a.k.a. context) are critical for contextual multi-armed bandits (MAB) performance. In practice of large scale online system, it is important to select and implement important features for the model: missing important features can led to sub-optimal reward outcome, and including irrelevant features can cause overfitting, poor model interpretability, and implementation cost. However, feature selection methods for conventional machine learning models fail short for contextual MAB use cases, as conventional methods select features correlated with the outcome variable, but not necessarily causing heterogeneuous treatment effect among arms which are truely important for contextual MAB. In this paper, we introduce model-free feature selection methods designed for contexutal MAB problem, based on heterogeneous causal effect contributed by the feature to the reward distribution. Empirical evaluation is conducted based on synthetic data as well as real data from an online experiment for optimizing content cover image in a recommender system. The results show this feature selection method effectively selects the important features that lead to higher contextual MAB reward than unimportant features. Compared with model embedded method, this model-free method has advantage of fast computation speed, ease of implementation, and prune of model mis-specification issues.

1. Introduction

Contextual multi-armed bandits (CMAB) (Tang etÂ al., 2014b, 2015) have emerged as a pivotal framework for optimizing decision-making in dynamic environments, particularly in personalization for recommender systems. Selecting the most relevant features is crucial to ensuring optimal model performance, improving interpretability and lower implementation costs.

Traditional feature selection methods (BolÃ³n-Canedo etÂ al., 2013; Chandrashekar and Sahin, 2014; Tang etÂ al., 2014a) focus on identifying features that are strongly correlated with the outcome variable. However, these methods may not be directly applicable to contextual MAB, where the goal is to identify features that contribute to heterogeneous treatment effects among different arms. In this context, the features that drive differential best actions given the context are more important than those that merely predict the average outcome.

Li et al. (Li etÂ al., 2010) introduced a model-based approach for feature selection in contextual MABs, where feature importance is inferred by the reward from model using such feature. Zhao et al.(Zhao etÂ al., 2022) proposed feature selection methods for HTE for uplift models (Olaya etÂ al., 2020; Rzepakowski and Jaroszewicz, 2012) with one treatment and one control, that has limitation for multiple arms.

In this paper, we propose novel, model-free feature selection methods for contextual MABs. The advantage of the proposed method includes:

  • â€¢

    It calculates the feature importance based on the incremental value and distribution divergence brought by the feature to the reward, instead of conventional correlation to outcome, therefore leading to better outcome in contextual MAB.

  • â€¢

    It considers nonlinear relationship between the feature and the heterogeneous causal effect among the arms.

  • â€¢

    It adapts to both continuous feature and categorical feature.

  • â€¢

    It is model free that is prune of model mis-specification.

  • â€¢

    It has a fast computation speed, especially compared with model-based methods.

We demonstrate the effectiveness of the proposed method through empirical evaluations using both simulated data and real-world data from online experiments. The results indicate that this approach not only enhances the performance of contextual MAB models but also offers advantages in terms of computational efficiency and robustness against model mis-specification.

2. Feature Selection Method

In the context of contextual Multi-Armed Bandit (MAB) problems, a feature is deemed important if it leads to heterogeneous treatment effects (HTE) across different arms. This means that the distribution of rewards from the arms varies depending on the values of the feature. Specifically, a feature is considered crucial if it causes the optimal arm to change across different ranges of the feature values, allowing the contextual MAB to optimize rewards based on the featureâ€™s context.

Figure Â 1 illustrates the distinction between correlational features and HTE features. Correlational features, while predictive of the outcome variable, do not affect the relative rewards of the arms across different feature values. In contrast, HTE features are valuable to the CMAB problem because they influence reward distributions and determine the best arm based on varying feature values.

We introduce two methods of calculating feature important scores: one based on the contribution to heterogeneous winning arms given the feature and the other based on the distribution divergence heterogeneity due to the feature. These two scores can be used separately or in combination.

3. Evaluation

3.1. Evaluation with Synthetic Data

The synthetic data is generated by the heterogeneous treatment effect data generator in the CausalML Python package (Chen etÂ al., 2020) (Zhao etÂ al., 2022), where the true feature importance is known. 10101010 trials are run, each trial with 50000500005000050000 sample, 3333 arms, and 10101010 features (5555 true important features for HTE effect, 2222 features correlational to outcome variable but not causing HTE, and 3333 features irrelevant to HTE and outcome). As shown in Figure 2, the proposed feature selection methods (HIE, HDD, and the combined score) effectively select the true important features that produce high CMAB rewards, which are evaluated by LinUCB, Nonlinear LinUCB (with quadratic term of the feature) and Cohort-based Thompson Sampling MAB.

3.2. Evaluation with Real Data

The real data is collected from an online randomized experiment. This experiment divided users into four variants, and serve a different content cover image in each variant. 600000600000600000600000 sample are used for evaluating the feature selection method, and CMAB reward is calculated using offline matching method (counting the reward based on matched case where the model selected arm equals the the observed arm). This real data evaluation shows the feature importance scores align with the CMAB rewards (NonLinear LinUCB not included as most features are categorical), and the one added irrelevant random feature is ranked the lowest.

Comparing HIE and HDD scores, HIE score is more sensitive - dropping to the lowest for unimportant features.

4. Future Work

This is a work in progress, and we are launching an online experiment with four variants, each variant implement a CMAB with a different feature, for more than ten thousand content (each content has a CMAB with multiple cover images to optimize). We will evaluate whether features with high feature importance score leads to better online CMAB reward.
